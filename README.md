# Counter App

A React Native counter app that lets you track multiple things at once with charts, goals, and customization.

## What It Does

Create unlimited counters for tracking anything - gym reps, water intake, daily habits, inventory, expenses, whatever you need. Each counter is fully customizable and comes with analytics to see your progress over time.

## Main Features

**Counter Management**
- Create as many counters as you want with custom names, icons, and colors
- Set custom increment/decrement values (count by 1, 5, 10, or any number)
- Organize counters into categories
- Set min/max limits and get alerts when you hit them
- Swipe right to increment, left to decrement
- Long press for quick actions
- Undo/redo your last 10 actions per counter

**Analytics Dashboard**
- See charts showing your counter history (7, 30, or 90 days)
- Track streaks and daily changes
- View total increments/decrements and averages

**Goal Setting**
- Set target values for any counter
- Visual progress bars
- Get notified when you reach your goals

**Customization**
- Light, dark, and OLED black themes


## Tech Stack

React Native with Redux/Zustand for state, React Navigation, Reanimated for animations, Victory Native for charts.

## Getting Started

```bash
npm install
npx pod-install # iOS only
npm run android # or npm run ios
```
